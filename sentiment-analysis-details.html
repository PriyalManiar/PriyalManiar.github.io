Comparitive Analysis of DL models for Text Sentiment Classification

  • Tech Stack:
    • Programming Language and Tools : Python, Jupyter Notebooks
    • Libraries and Frameworks : Scikit-Learn, Tensorflow, Keras, NLTK, Seaborn
  • Concepts:
    • NLP Text Preprocessing : Tokenisation, Stemming, Lemmatization, Stop Word Removal
    • Deep Learning Models: CNN, RCNN, LSTM, BiLSTM, BERT
    • Hyperparameter Tuning & Model Evaluation
    • Text Sentiment Classification

Project Overview

  • A comparitive analysis of the performance of various deep learning models in text sentiment analysis across 7 sentiment classes (joy, anger, fear, shame, disgust, sadness, guilt) in the ISEAR dataset.

  • Implemented DL architectures - CNN, RCNN, LSTM, BiLSTM and BERT for sentiment classification.

  • Optimised model performance using hyperparameter tuning, reducing loss by 3% and achieving model accuracy of 86%.
