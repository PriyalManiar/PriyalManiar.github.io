HPV Prevalence Trends and Vaccination Coverage Analysis

  • Tech Stack
    • Data Analysis and ETL : Python, Excel, Tableau Prep
    • Data Visualization : Tableau
  • Github Link: HPV Analysis Dashboards

Project Overview

A data-driven analysis of global HPV trends, leveraging data from the HPV Center Repository and CDC across 40 countries to identify risk factors, demographic variations, and regional disparities. The dashboards present insights to optimize healthcare strategies and interventions.

Dashboard Description
1. HPV Trends : Incidence, Mortality and Prevalence Analysis
2. Vaccination and Population Trends for Under 15 Age Group
3. USA HPV Vaccination Coverage : Vaccine uptake based on geography, gender, and socioeconomic factors.
4. HPV Preventive Strategies : Cervical Screening and Vacccination Adoption Analysis
